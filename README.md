# クレジット文ジェネレーター（無料Web版）

動画クリエイター向けの無料ツール。使った素材（BGM・効果音・イラスト等）のサイトと素材名を入れるだけで、概要欄にそのまま貼れるクレジット表記を自動生成する。

有料デスクトップ版（素材ライセンス台帳・コンテンツ紐づけ・警告エンジン・案件管理）への導線となるリードマグネット。

## 使い方（ローカル確認）

`index.html` をブラウザで開くだけ（サーバー不要、file:// 直開きで動作）。

## ファイル構成

- `index.html` — ページ本体（ツールUI＋対応サイト一覧＋FAQ＋免責）
- `rules.js` — 素材サイトごとの表記ルールデータ（★ここを編集してサイト追加）
- `app.js` — 生成・コピー・テーブル描画のロジック
- `style.css` — スタイル（ライト/ダーク自動対応）
- `articles/` — SEO記事（credit-rules.html＝表記ルール一覧記事）
- `sitemap.xml` — サイトマップ（記事追加時はここにもURLを足す）

## サイトの追加方法

`rules.js` の `SITE_RULES` 配列にオブジェクトを1つ追加する。

```js
{
  id: "一意のID",
  name: "セレクトボックスに出す名前",
  label: "クレジット行に出す名前",
  suffix: "（CC BY 4.0）など、なければ空文字",
  url: "クレジットに載せるURL",
  ruleUrl: "公式規約ページのURL",
  policy: "required | optional | none | conditional",
  note: "UIに表示する注意書き。『注意：』を含めると生成後の確認事項にも出る",
  verified: "規約を確認した日付"
}
```
